# 🍽️ Restaurant Data Analysis Using Python

## 📌 Project Overview
This project focuses on analyzing a restaurant dataset using *Python* and popular data science libraries.  
The main objective was to explore restaurant ratings, cuisines, locations, booking facilities, delivery services, and pricing trends to generate meaningful business insights.

## 🎯 Objectives
- Perform data cleaning and preprocessing  
- Handle missing values  
- Explore restaurant ratings and customer trends  
- Analyze top cities and cuisines  
- Study table booking and online delivery availability  
- Compare price ranges with ratings  
- Create useful visualizations  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## 📂 Dataset Information
- Total Rows: *9551*  
- Total Columns: *21*  

## 📊 Tasks Performed

### 🔹 Level 1
- Data Exploration and Preprocessing  
- Descriptive Analysis  
- Geospatial Analysis  

### 🔹 Level 2
- Table Booking & Online Delivery Analysis  
- Price Range Analysis  
- Feature Engineering  

## 📈 Key Insights
- Most restaurants are concentrated in major cities.  
- Some cuisines are more popular than others.  
- Restaurants with table booking tend to have better ratings.  
- Mid-range priced restaurants are most common.  
- Online delivery is available only in selected restaurants.  

## 📷 Visualizations Included
- Aggregate Rating Distribution  
- Top 10 Cities with Restaurants  
- Top 10 Cuisines  
- Price Range vs Average Rating  
- Table Booking Availability  

## 📌 Conclusion
This project demonstrates practical use of *data cleaning, exploratory data analysis, feature engineering, and visualization* techniques to solve real-world business problems using restaurant data.
